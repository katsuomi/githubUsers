## 🔰 Install & Setup

#### 2. Getting source code

```bash
git clone https://github.com/katsuomi/githubUsers.git
cd githubUsers
```

#### 3. Start-up

The following procedure, start the container. 

```bash
# package install 
$ yarn install 

# start local server
$ yarn start


# If the following local server started, it's ok. 
# http://localhost:3000
```
